# Veckouppgift 1 - Test Automation Course

## Project Structure

This project contains solutions for Veckouppgift 1 (Week Assignment 1) from the Python Programming course.

### Files:

- **main.py** - Section 1: Hello World program
- **main_2.py** - Section 2: Code review exercise (bug fixing)
- **main_3.py** - Section 3: Variables and data types (requires user input)
- **main_4.py** - Section 4: Advanced exercises (requires user input)

## How to Run Each File

### Section 1 & 2 (No input required):
```powershell
python main.py
python main_2.py
```

### Section 3 (User input required):
```powershell
python main_3.py
```
**What it does:**
- Asks for two integers and calculates their sum
- Calculates jacket prices with discounts

**Test examples:**
- Enter: 15 and 25 → Sum should be: 40
- Enter: 10% discount → Final price: 1800 kr

### Section 4 (User input required):
```powershell
python main_4.py
```
**What it does:**
1. **Travel time calculator** - Stockholm to Göteborg (470 km)
   - Test: Enter 100 km/h → Should take 4.7 hours or 4 hours 42 minutes
   
2. **Pythagoras theorem** - Calculates hypotenuse
   - Test: Enter 3 and 4 → Should give 5.0
   
3. **Date calculator** - Shows today's date and date in 7 days
